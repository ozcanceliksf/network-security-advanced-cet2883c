# Network Security Advanced — CET2883C
**CET2883C · Santa Fe College · Fall 2025**  
**Certification Alignment:** CompTIA Security+ / Network+

Advanced network security course focusing on real-world enterprise security configurations, network defense strategies, and security infrastructure management.

---

## 📚 Topics Covered

### 🔥 Firewall & Access Control
- Firewall configuration and rule management
- Access Control Lists (ACLs)
- Stateful vs stateless inspection
- Next-Generation Firewall (NGFW) concepts

### 🔒 VPN & Remote Access Security
- VPN types: Site-to-site, Remote access, SSL VPN
- IPSec protocol suite — AH, ESP, IKE
- Tunneling protocols: L2TP, PPTP, OpenVPN
- Split tunneling and full tunnel configurations

### 🔑 Authentication & Identity
- RADIUS authentication server configuration
- TACACS+ vs RADIUS comparison
- 802.1X network access control
- Multi-factor authentication (MFA) implementation

### 🌐 Secure Network Design
- DMZ architecture and design
- Network segmentation strategies
- VLAN security and trunking
- Defense-in-depth model

### 📡 Network Monitoring & Defense
- IDS vs IPS — detection and prevention
- Network traffic analysis
- Log management and SIEM fundamentals
- Honeypots and deception technology

### 🛡️ Advanced Security Concepts
- Public Key Infrastructure (PKI)
- Certificate management and SSL/TLS
- Network forensics fundamentals
- Incident response procedures

```

Auth Protocols:    RADIUS · TACACS+ · Kerberos · LDAP

Defense Layers:    Perimeter · Network · Host · Application · Data
```
